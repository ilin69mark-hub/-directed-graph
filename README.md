# Ориентированный граф с поиском кратчайших путей

Это приложение реализует ориентированный граф и позволяет находить кратчайшие пути между любыми парами вершин с использованием алгоритма Флойда-Уоршелла.

## Структура проекта

- `main.go` - основной файл с реализацией ориентированного графа и алгоритмов поиска кратчайших путей

## Особенности реализации

- Реализация ориентированного графа с использованием матрицы смежности
- Алгоритм Флойда-Уоршелла для нахождения кратчайших путей между всеми парами вершин
- Поддержка взвешенных ребер
- Возможность добавления направленных ребер между вершинами
- Функция получения кратчайшего расстояния между двумя конкретными вершинами

## Как запустить

1. Убедитесь, что у вас установлен Go (версия 1.16 или выше)
2. Склонируйте или скачайте этот проект
3. Перейдите в директорию проекта
4. Выполните команду:

```bash
go run main.go
```

Или вы можете собрать бинарный файл и затем запустить его:

```bash
go build -o graph_app main.go
./graph_app
```

## Пример работы

Программа создает граф с 4 вершинами и следующими ребрами:
- Ребро из 0 в 1 с весом 5
- Ребро из 0 в 2 с весом 3
- Ребро из 1 в 2 с весом 2
- Ребро из 1 в 3 с весом 6
- Ребро из 2 в 3 с весом 7

Затем программа выводит матрицу смежности графа и кратчайшие пути между всеми парами вершин.
